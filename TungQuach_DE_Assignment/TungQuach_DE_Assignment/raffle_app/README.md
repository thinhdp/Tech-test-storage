# Raffle App
Refer to [Problem statement](problem-statement.md) for requirements for Raffle App

# How to
## Install this project
On Linux
```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```  
On Windows
```shell
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```  
## Run this project
```bash
python main.py
```
## Run unit testing
```bash
python -m pytest
```
